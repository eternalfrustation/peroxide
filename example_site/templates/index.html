Another platform for hosting blogs.

Features

Free
Unlike some other "Platforms", It is completely free
Open Source
Now, you can skip the diving into proprietary docs and dive into equally unmaintaned docs of open source software.
Performant
Currently untested, but i can say with relative confidence that a 1 to 1 instance of peroxide will outperform PHP based alternatives
Easy to use
Uses YAML and standard html for configuration, WYSIWYG tool planned.
Accidental Competitor to Neocities
As I am writing this, I have also realized that this might also end up competing/taking load off of Neocities.

Prerequisites

First, you may need some things preinstalled
cargo, git

Install

$ git clone https://github.com/eternalfrustation/peroxide.git
$ cd peroxide
$ cargo install .
